클라우드 배포

Azure App Service를 활용한 배포·스케일링·모니터링까지 한 번에 익힙니다.

추천 수강 로드맵

FAQ에서 가장 많이 묻는 “어떤 순서로 들어야 하나요?”에 대한 답입니다.

1. C# & .NET 기초

언어 문법, OOP, 컬렉션, LINQ, 비동기(Async/Await), .NET 런타임 이해.

필수초급

2. ASP.NET Core Fundamentals

Program/Startup, DI, 구성(Configuration), 환경, 로깅, 미들웨어 파이프라인.

핵심초·중급

3. MVC 패턴 & Razor Pages

라우팅, 컨트롤러/뷰, 모델 바인딩/검증, 태그 헬퍼, Razor Pages로 CRUD 구축.

웹 기초중급

4. Entity Framework Core (DB)

Code-First/DB-First, 마이그레이션, 관계 매핑, LINQ 쿼리, 성능/추적 제어.

실무중급

5. 인증/인가 & Identity

쿠키/토큰, ASP.NET Core Identity, 외부 로그인, 역할/정책 기반 권한.

보안중급

6. Web API & Swagger

REST 설계, 모델 유효성, 버저닝, Swagger/OpenAPI, 필터, 예외 처리.

실무중급

7. Blazor (Server & WebAssembly)

상태 관리, 양방향 바인딩, Form, 인증 연계, 컴포넌트 아키텍처.

프론트엔드중급

8. 고급: 미들웨어/필터/성능/캐싱

고성능 엔드포인트, 응답 캐싱, 분산 캐시, 정책 캐시, 병목 분석.

고급중·고급

9. 로깅/모니터링 (Serilog 등)

구조적 로깅, 분산 추적, 대시보드, 예외/성능 모니터링.

운영중급

10. 배포 & Azure App Service

IIS/리눅스/Nginx 배포, Azure App Service 배포·스케일·슬롯·로그 스트림.

클라우드필수

11. CI/CD (GitHub Actions, Azure Pipelines)

빌드/테스트/배포 자동화, 비밀관리(Key Vault), 단계적 배포(Blue/Green).

DevOps중·고급

12. 실전 프로젝트 & 리팩터링

도메인 설계, 모듈화, 테스트, 성능 최적화, 보안 점검, 문서화.

완주포트폴리오

FAQ

자주 묻는 질문

C# & .NET 기초 → ASP.NET Core Fundamentals → MVC/Razor Pages 순서를 추천합니다. 이후 EF Core와 Identity를 통해 데이터/보안을 익히세요.
MVC/Razor Pages 이후 Web API를 학습하고, Blazor(Server/WASM)로 프론트엔드까지 확장하세요.
Azure App Service로 배포/스케일링을 실습하고, Serilog와 모니터링으로 운영 품질을 높입니다. CI/CD는 GitHub Actions 또는 Azure Pipelines를 추천합니다.
입문 C#/.NET → 핵심 ASP.NET Core 기본 → 웹 MVC/Razor → DB EF Core → 보안 Identity → API Web API/Swagger → UI Blazor → 운영 성능/로깅 → 배포 Azure App Service → DevOps CI/CD → 완주 실전 프로젝트.
